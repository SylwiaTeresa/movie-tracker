# 🎬 Movie Tracker

A simple and responsive movie tracking application built with React Native, Expo and TypeScript.

## ✨ Features

- Add movies to your watch list
- Mark movies as watched
- Undo watched status
- Rate watched movies from 1–5 stars
- Remove movies with confirmation alert
- Track how many movies you have watched
- Bottom tab navigation
- Global state management with Context API

---

## 🛠️ Built With

- React Native

- Expo
- React Navigation
- Context API

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/SylwiaTeresa/movie-tracker.git
```

Install dependencies:

```bash
npm install
```

Start the Expo development server:

```bash
npx expo start
```

---

## 📱 Run on specific platforms

```bash
npm run ios
```

```bash
npm run android
```

```bash
npm run web
```
